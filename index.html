Google Review Helper

Enter your Google Places API Key and a business name or Maps link to retrieve a Google Review link, as well as QR Codes pointing to the Review Page and the Maps Business Page ands its Place ID if you need it for something other than the direct Google Review link.

The Google Places API has a generous free monthly credit of up to 10,000 monthly API calls, meaning for the occasional lookup, this service is basically free to use. However, they do still require you to set up a Google Developer Account with a valid billing account in order to use this service. If you're using this service to help clients get Google Reviews, then the small amount of work necessary to set everything up is well worth it for what you can offer them.

Your API key is saved locally in your browser and never sent anywhere except back to Google.
